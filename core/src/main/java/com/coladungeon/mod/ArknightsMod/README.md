## 具体项 干员相关
干员本体有三种存在形式：被部署，作为NPC存在，和开局职业（讲述者，肉鸽的那些）
NPC另说了

部署需要在命令终端消耗希望进行记录之后消耗部署费用。
部署后可以切换指令，来切换AI状态。（指令可能有不同，行动方式也不同）
部署后有背包，可以使用一些武器（但除了规定的武器，不能额外搞事情。
强度受命令终端影响。可以接收一些物品来额外增强

开局职业就没什么好说了，也有终端相关天赋，但是应该不多

  你知道的.jpg
  - 处决者
    使用武器进行普攻（伏击），神器在隐身之后触发落地效果(也不一定)
  - 推击手
    武器推击
    **已实现：**
    - PusherWeapon：推击手武器，可以将敌人推开并造成伤害
    - 推击：将敌人推开1格并造成伤害
    - 蓄力推击：消耗充能进行强力推击，最多推3格
    - 攻击时有30%概率获得充能
    - 已集成到ArkSupply中，可直接获得完整装备套装
  - 钩索师
    武器可拉近，或拉近自身到指定方块
    **已实现：**
    - HookWeapon：钩索师武器，可以拉近敌人或拉近自身
    - 钩索：将3格内的敌人拉近并造成眩晕
    - 拉近自身：消耗充能移动到附近位置
    - 攻击时有20%概率获得充能
    - 已集成到ArkSupply中，可直接获得完整装备套装
  - 陷阱师
    神器，产生陷阱，可以加材料整活
    **已实现：**
    - TrappistWeapon：包含陷阱师工具箱神器和爆炸陷阱的完整实现
    - 陷阱师工具箱神器：可以消耗充能制作爆炸陷阱
    - 爆炸陷阱：可投掷的陷阱，当敌人靠近时自动爆炸，对周围敌人造成伤害
    - 已集成到ArkSupply中，可直接获得完整装备套装
  - 怪杰
    携带会自动掉血，攻击给自身加buff，
    **已实现：**
    - GeekWeapon：怪杰武器，携带会持续掉血但攻击时获得增益
    - 混沌之力：获得随机buff（加速、隐身、狂怒），增加混沌等级
    - 献祭：消耗所有混沌等级获得强力效果
    - 携带时每10回合掉血，攻击时有概率获得buff
    - 已集成到ArkSupply中，可直接获得完整装备套装
  - 行商
    携带会掉cost，有特殊技能
    **已实现：**
    - MerchantWeapon：行商武器，携带会消耗cost但可以进行交易
    - 交易：消耗1点cost获得随机物品
    - 谈判：消耗交易等级获得大量cost
    - 携带时每15回合消耗1点cost，攻击时有概率增加交易等级
    - 已集成到ArkSupply中，可直接获得完整装备套装
  - 傀儡师
    神器替死，和武器有联动
    **已实现：**
    - DollkeeperWeapon：人偶师武器，可以召唤人偶并与之联动
    - 召唤人偶：召唤人偶神器，提供替死功能
    - 操控人偶：消耗人偶等级获得强力效果
    - 攻击时有概率增加人偶等级，有人偶时攻击力提升20%
    - 替死机制：人偶在角色死亡时阻挡死亡，给延缓死亡buff，不能主动攻击，时间过去后恢复一半血量或真死亡
    - 已集成到ArkSupply中，可直接获得完整装备套装
  - 炼金师
    与工匠类似，不过造的是投射物
    **已实现：**
    - AlchemistWeapon：炼金师武器，可以炼制各种炼金物品
    - 炼制炼金物品：创建随机炼金物品（治疗、力量、隐形、火焰、冰冻）
    - 强化炼金物品：消耗炼金等级强化现有炼金物品
    - 攻击时有概率增加炼金等级
    - 已集成到ArkSupply中，可直接获得完整装备套装
  - 巡空者
    神器，浮空时有特殊功能
    **已实现：**
    - SkyRangerWeapon：巡空者武器，可以召唤羽翼并在空中战斗
    - 召唤羽翼：召唤羽翼神器，提供飞行能力
    - 空袭：消耗飞行等级获得飞行和加速效果
    - 攻击时有概率增加飞行等级，飞行时攻击力提升30%
